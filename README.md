# Object Annotation Using CVAT

Dataset ini berisi anotasi *object detection* yang dibuat menggunakan CVAT dan diekspor dalam format **COCO 1.0**.  
Setiap objek diberi *bounding box* dan label kategori sesuai kebutuhan proyek.
Format ini umum digunakan untuk melatih berbagai model deteksi objek, untuk kebutuhan pembelajaran mesin atau pengembangan aplikasi berbasis AI.
